 安装及使用说明

1. 环境要求
   - Python 3.x
   - Tkinter库（通常随Python安装包一起提供）

2. 安装步骤

   a. 克隆或下载项目
   
   git clone <repository_url>
   
   或者直接下载并解压项目压缩包。

   b. 安装依赖库
   打开命令行工具（如命令提示符或终端），导航到项目目录`雷子豪-20223004514-数据结构课程设计/`，并运行以下命令以安装所需的库：
   
   pip install tkinter
   

3. 运行程序
   进入项目目录`HashTableProject/`，运行主程序文件：
   
   python hash.py
   

4. 程序使用说明
   - 启动程序后，会显示一个图形化界面，包括哈希表大小输入框、元素输入框及插入、查找、删除按钮。
   - 在“哈希表大小”输入框中输入正整数并点击“创建哈希表”按钮，初始化哈希表。
   - 在“元素”输入框中输入整数，然后点击“插入”按钮，将元素插入到哈希表。
   - 点击“查找”按钮，查找哈希表中的元素。
   - 点击“删除”按钮，删除哈希表中的元素。
   - 每次操作后，哈希表的显示区域会自动更新。
